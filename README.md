Calculator.py: This Python program is a command-line interface (CLI) application that performs basic arithmetic operations. It prompts the user to input two integers and select a mathematical operator, then calculates and displays the result.
-----------------------------------------------------------------------------------------------
Create(2).py: 100 Birds of the World is an interactive, Python-based program designed to help you discover beautiful bird species based on their appearance and diet. Whether you are a casual bird-watcher or an aspiring ornithologist, this tool brings the fascinating world of birds right to your screen.
-----------------------------------------------------------------------------------------------
Fable(1).py: This Python script simulates a classic, randomized race between a Tortoise and a Hare to a finish line of \(50\) steps. It uses the random library to model each character's daily progress and the Hare's chance of randomly falling asleep, outputting the race's step-by-step progress.
-----------------------------------------------------------------------------------------------
gam3.py: This is a Python-based Random Number Guessing Game that offers multiple difficulty levels (Easy, Medium, and Hard) to challenge the player's logic and speed.
-----------------------------------------------------------------------------------------------
madlibs(1).py: This Python script is an interactive Mad Libs game that prompts users to input words or select random options to generate a hilarious, customized story about a disastrous night out with friends, complete with DoorDash food and a random movie.
------------------------------------------------------------------------------------------------
